# 관리왕 — 서비스 소개 & 마케팅 피치

> 버전: 2026-04-07  
> 대상: 영업팀, 마케팅팀, 잠재 고객 제안서 작성 시 참고

---

## 한 줄 소개

> **"복잡한 인사 관리, 이제 관리왕 하나로 끝."**  
> 중소사업장을 위한 올인원 직원 관리 플랫폼. 출퇴근부터 급여·계약·세무까지, 법 지키면서 쉽게.

---

## 왜 관리왕인가?

### ❌ 기존 방식의 문제

| 문제 | 현실 |
|------|------|
| 엑셀로 출퇴근 기록 | 오기입, 분실, 관리 인력 낭비 |
| 급여는 사장님 머릿속 | 분쟁 발생 시 증거 없음 |
| 세무 신고 깜빡 | 가산세 폭탄 |
| 직원마다 다른 근무 시간 | 수동 계산 지옥 |
| 인사 정보가 여러 툴에 분산 | 전체 현황 파악 불가 |

### ✅ 관리왕의 해답

| 기존 문제 | 관리왕 해결책 |
|----------|-------------|
| 출퇴근 기록 오류 | 다중 방식 자동 기록 (앱·GPS·QR·WiFi) |
| 급여 분쟁 | 법적 효력 있는 급여 명세서, 암호화 보관 |
| 세무 신고 누락 | 35일 이내 세무 할 일 자동 알림 |
| 개인별 다른 근무시간 | 계약서 기반 개인 스케줄 + 법정 휴게시간 자동 계산 |
| 정보 분산 | 하나의 플랫폼에서 전부 관리 |

## 핵심 기능 — 영업 포인트 10가지

---

### 1. 📱 다중 출퇴근 방식 — "어디서든, 어떻게든 찍힌다"

**5가지 방식을 동시에 활성화, 먼저 처리된 게 기록됨.**

| 방식 | 설명 |
|------|------|
| 클릭 | 앱/웹에서 버튼 한 번 |
| GPS | 사업장 반경 안에서 자동 인식 |
| WiFi | 사내 와이파이 연결되면 출근 |
| QR | 관리자 화면의 QR을 스캔 |
| 생체인증 | 지문/Face ID 인증 후 처리 |

> **경쟁사 대비**: 대부분의 경쟁 앱은 1~2가지 방식만 지원.  
> 관리왕은 5가지를 **동시에 켜놓고** 가장 먼저 인식된 방식으로 자동 처리.

**사장님이 좋아하는 이유**:
- 현장직(GPS) + 재택직(클릭) + 사무직(WiFi)이 섞여 있어도 하나의 시스템으로 OK
- QR 방식: 타임카드 기계 없이도 출퇴근 스탬프 효과

---

### 2. ⚖️ 법정 휴게시간 자동 계산 — "근로기준법을 몰라도 된다"

**근로기준법 제54조 자동 준수.**

```
총 근무 4시간 이상 → 30분 휴게 자동 차감
총 근무 8시간 이상 → 60분 휴게 자동 차감
```

> 관리자가 법을 몰라도, 시스템이 알아서 실 근무시간을 계산합니다.  
> **법적 분쟁 시 "적법하게 산정됨"을 시스템이 증명.**

**사장님이 좋아하는 이유**:
- 근로기준법 위반 걱정 없음
- 연장근로 수당 계산 기준이 정확해짐

---

### 3. 🕐 계약 기반 개인 근무 스케줄 — "각자 다른 출퇴근 시간, 자동으로"

- 직원마다 계약서에 따라 출퇴근 시간이 다를 수 있음
- 개인 스케줄을 등록하면 **지각 판정·실 근무시간 모두 개인 기준으로 자동 계산**
- 탄력근무제, 시차 출퇴근제 완벽 지원

```
A 직원: 09:00 ~ 18:00 (일반)
B 직원: 10:00 ~ 19:00 (시차 출퇴근)
C 직원: 07:00 ~ 16:00 (현장 조기 출근)
→ 각자 다른 기준으로 지각 여부 자동 판정
```

**변경 절차도 안전하게**:
- Owner는 즉시 변경
- 일반 관리자는 전자 결재 → 승인 후 자동 적용

---

### 4. 🔐 세분화된 정보 접근 권한 — "알 사람만 알아야 할 정보"

**급여 정보는 재무팀만, HR 노트는 인사팀만.**

```
급여 정보 열람 권한
  ├── Owner: 전체 열람 (항상)
  ├── 재무/회계/경리 부서 관리자: 설정에 따라 열람
  ├── 명시적 권한 부여된 자: 허용된 범위만
  └── 그 외: 403 접근 차단

HR 노트 열람 권한
  ├── Owner: 전체 열람
  ├── 인사/HR 부서 관리자: 설정에 따라 열람
  └── 그 외: 403 접근 차단
```

> 권한 변경 자체도 **결재 기안**으로 이력 남김 → 감사(audit) 추적 가능.

**사장님이 좋아하는 이유**:
- 급여 정보가 새지 않음
- 누가, 언제, 무엇을 바꿨는지 기록됨

---

### 5. 📋 전자 결재 시스템 — "종이 없이, 흔적은 남겨"

- 휴가, 지출, 출장, 연장근무, 인사 발령
- **다단계 결재선** 지원 (팀장 → 부장 → 대표)
- **자동 처리 연동**: 결재 완료 시 시스템이 자동으로 권한/스케줄 적용
- 모바일에서도 결재 가능

> 결재 완료 버튼 하나로 → DB 자동 업데이트 → 알림 발송까지 자동.

---

### 6. 🧾 세무·노무 자동 알림 — "신고 기한을 절대 놓치지 않는다"

- 35일 이내 세무·노무 처리 항목 **자동 목록화**
- 4대보험 취득/상실 신고서 자동 생성
- 원천징수이행상황 HTML 출력
- 연말정산 자료 정리
- 퇴직금 자동 계산
- **D-7, D-3 자동 알림** (이메일 + 인앱)

> ⚠️ 모든 세무 자료는 참고용이며, 최종 신고는 공인 세무사 확인을 권장합니다.

---

### 7. 🎨 브랜딩 커스터마이징 — "우리 회사 색깔 그대로"

**Free 플랜부터 브랜딩 가능.**

- 회사 로고 업로드
- 커버 이미지 (PC/모바일 각각)
- 브랜딩 텍스트 색상

> **경쟁사 대비 최대 차별점**:  
> 대부분의 경쟁사는 로고·커스텀 UI를 **월 수백만 원짜리 Enterprise 플랜**에서만 제공.  
> 관리왕은 **무료 플랜부터** 제공 → 직원이 앱을 열 때마다 "우리 회사" 느낌.

---

### 8. 🤖 AI 업무 템플릿 추천 — "제목만 쓰면 내용이 채워진다"

**35가지 업무 카테고리 × AI 자동 분류.**

| 카테고리 그룹 | 포함 업무 유형 |
|-------------|--------------|
| 보고·기획 | 주간보고, 월간보고, 기획안, 제안서, 회의, 발표, 리서치 |
| HR·인사 | 채용, 교육, 인사평가, 온보딩 |
| 재무·회계 | 회계정산, 예산, 청구서, 급여 |
| 영업·마케팅 | 영업, 고객관리, 마케팅, SNS, 콘텐츠, 행사 |
| 개발·IT | 개발, 테스트, 배포, 보안 |
| 계약·법무 | 계약, 법무, 컴플라이언스 |
| 운영·총무 | 구매, 물류, 시설, 문서관리 |
| 고객지원 | CS, 대외 커뮤니케이션 |

1. 업무 제목 입력
2. "AI 추천" 버튼 클릭 → AI가 제목 분석 후 카테고리 자동 분류
3. 해당 카테고리 마크다운 템플릿 즉시 로드

> 담당자 선택 시 **프로필 사진이 이름 옆에 표시**되어 팀원을 쉽게 식별.

---

### 9. 📋 기존 업무 복제 — "똑같은 업무, 한 번만 만들면 된다"

반복 업무(주간보고, 월간 정산 등)를 매번 새로 만들 필요 없이:
- 기존 업무 목록에서 선택 → 내용 그대로 복사
- 제목·마감일만 수정해서 즉시 생성

---

### 10. 🔒 업무 삭제 2단계 승인 — "실수로 지워지는 업무 없음"

진행중·검토중·완료 업무는 단독으로 삭제 불가:

```
관리자가 삭제 요청 → 담당자가 최종 승인 후 삭제
담당자가 삭제 요청 → 관리자가 최종 승인 후 삭제
```

> 업무 삭제 이력이 남아 감사(audit) 추적 가능.  
> 실수·의도치 않은 삭제로 인한 업무 유실 완전 차단.

## 고객 시나리오 — "이런 분께 딱입니다"

### 시나리오 1: 소규모 제조업체 (30명)
> "생산직은 7시 출근, 사무직은 9시 출근인데 다 섞여 있어서 관리가 힘들어요."

→ **개인 근무 스케줄** 등록 → 각자 기준으로 지각 판정, 실 근무시간 자동 계산  
→ 현장직은 **QR** 방식, 사무직은 **클릭** 방식 동시 활성화

---

### 시나리오 2: IT 스타트업 (15명, 재택 포함)
> "재택이 많아서 출퇴근 기록이 제각각이에요. 그리고 팀마다 급여를 숨겨야 해요."

→ **클릭 + GPS** 방식으로 장소 무관 출퇴근  
→ **급여 접근 권한**으로 재무팀만 급여 열람 가능  
→ **탄력근무제** 개인 스케줄 설정

---

### 시나리오 3: 요식업 프랜차이즈 본사 (60명)
> "매장마다 근무 시간이 달라요. 세금 신고 날짜도 맨날 헷갈려요."

→ **매장별 근무 시간 차등 설정**  
→ 세무 할 일 자동 목록 + **D-7 알림**  
→ 4대보험 취득/상실 신고서 자동 생성

```
✅ 이메일·이름 AES-256-GCM 암호화 저장

```

## FAQ (영업 현장 대응)

**Q: 기존에 쓰던 엑셀 데이터를 옮길 수 있나요?**  
A: 직원 일괄 초대(이메일/링크)로 빠르게 온보딩 가능합니다. 데이터 이전 지원은 Pro 이상 문의.

**Q: 모바일 앱이 있나요?**  
A: iOS/Android 모두 지원합니다 (Expo React Native 기반).

**Q: 우리 회사 로고를 앱에 넣을 수 있나요?**  
A: 네, Free 플랜부터 로고·커버 이미지 설정이 가능합니다.

**Q: 세무 자동 계산 결과를 그대로 신고해도 되나요?**  
A: 관리왕의 세무 계산은 **참고용**입니다. 최종 신고는 담당 세무사와 확인하시기 바랍니다.

**Q: 직원이 GPS를 끄면 출퇴근이 안 되나요?**  
A: GPS를 끄더라도 클릭·QR·WiFi 등 다른 방식으로 출퇴근할 수 있습니다.  
GPS 기록은 "범위 밖 플래그"만 남기며, 출퇴근 자체를 막지 않습니다.

**Q: 급여 정보가 외부로 새어 나가지 않나요?**  
A: 급여 정보는 재무/회계팀 등 명시적으로 권한을 부여한 사람만 열람할 수 있습니다.  
권한 변경 자체도 결재 이력이 남습니다.
