# 데이터 활용 로드맵

> 작성일: 2026-03-31
> 전제: 관리왕 사용자가 지속적으로 데이터를 쌓으면서 활용할 수 있는 방향

---

## 현재 축적되는 데이터

| 영역 | 데이터 |
|------|--------|
| 근태 | 출퇴근 시간, GPS, 지각/결근, 근무시간 합계 |
| 급여 | 월급, 4대보험, 공제항목, 연봉 이력 |
| 휴가 | 잔여연차, 사용 패턴, 신청/승인 이력 |
| 계약 | 고용 형태, 계약 기간, 갱신 이력 |
| 평가 | 평가 점수, 상급자 의견, 주기별 추이 |
| 교육 | 수강 이력, 수료율, 교육 시간 |
| 결재 | 기안 유형, 처리 시간, 반려 사유 |
| 활동로그 | 로그인 패턴, 기능 사용 빈도 |

---

## 단계별 로드맵

### 지금 (데이터 수집 기반 다지기)
- 근태/급여/평가 데이터 정확히 쌓기
- 인덱스 + 백업으로 데이터 안전하게 보관
- 암호화 인프라 유지 (개인정보보호법)

### 100~500명 — 즉시 수익화 가능 (난이도 낮음)

#### 세금신고 자료 자동 생성 ★ 최우선
- 근로소득 원천징수 영수증 (급여 데이터 → PDF)
- 연말정산 간소화 자료 (급여 + 4대보험 집계)
- 사업주 인건비 신고서 (직원별 지급액 합산)
- 4대보험 취득/상실 신고서 (입퇴사 이벤트 기반)
- → 세무사/공인노무사 연결 서비스로 확장 가능 (B2B 파트너십)

#### 퇴직금 자동 계산
- 입사일 + 급여 이력 + 재직기간 → 퇴직금 정확히 계산
- 퇴직 처리 시 자동 계산서 생성

#### 주 52시간 위반 실시간 경보
- 실시간 주간 근무시간 합계 모니터링
- 위반 임박 시 관리자 알림

### 1,000명+ — AI 분석 기능 (난이도 중간)

#### 직원 이탈 위험도 예측
- 신호: 평가 점수 하락 + 연차 소진율 급증 + 로그인 빈도 감소 + 결재 처리 지연
- 위험도 점수화 → 관리자 알림

#### 급여 벤치마킹
- 익명화된 집계 데이터 활용
- 동종업계 + 같은 직급 평균 급여 비교
- 채용 시 적정 연봉 제안

#### 최적 채용 시점 예측
- 퇴직 패턴 분석 → 계절성 이탈 예측
- 계약 만료 예정자 기반 채용 시기 사전 알림

#### 생산성 패턴 리포트
- 요일별/시간대별 출근율, 초과근무 분포
- 부서별 연차 소진 패턴
- 근무 제도 개선 리포트 자동 생성

### 3,000명+ — 플랫폼화 (난이도 높음)

#### 노무 자동화 파트너 마켓플레이스
- 공인노무사 4대보험 신고 대행 연결
- 취업규칙 작성 지원
- 중개 수수료 수익 모델

#### 정부지원금 자동 매칭 ★
- 직원 구성 + 업종 + 고용 형태 분석
- 받을 수 있는 고용지원금 자동 탐색 (청년 고용 장려금, 일·가정 양립 지원금 등)
- 신청서 초안 자동 생성

#### 금융 서비스 연계
- 급여 데이터 기반 사업자 대출 심사 자료 자동 생성
- 직원 급여 선지급 서비스 (임금 핀테크)

#### 산업재해 리스크 관리
- 과로 누적 패턴 감지
- 산재보험 신고 자료 자동화

## 수익 모델 확장 방향

```
현재: SaaS 구독료
      ↓
단기: 세금/노무 문서 프리미엄 기능 (플랜 업셀)
      ↓
중기: 노무사/세무사 파트너 중개 수수료
      ↓
장기: 금융 서비스 연계 (데이터 기반 심사)
```
