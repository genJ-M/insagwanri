## 1. 회원가입 화면 `/register`

**레이아웃**: 로그인과 동일한 중앙 카드, 2단계 스텝

### Step 1 — 계정 정보
- 이름 (text)
- 이메일 (email)
- 비밀번호 (password) + 보기/숨기기
- 비밀번호 확인 (password)
- 회사명 (text)
- 「다음」 버튼 → Step 2

### Step 2 — 이메일 인증 안내
- "입력하신 이메일로 인증 메일을 발송했습니다"
- 이메일 아이콘 + 이메일 주소 표시
- 「재발송」 버튼 (쿨타임 60초)
- 이미 계정이 있으신가요? → 로그인 링크

**URL 흐름**
```
/register → (가입 후) → /register?step=verify → (인증 후) → / (대시보드)
```

## 2. 팀 관리 화면 `/team`

**레이아웃**: 상단 헤더 + 탭 + 테이블/리스트

### 상단
- 제목: "팀 관리"
- 「직원 초대」 버튼 (owner/manager만 표시)

### 탭
- 전체 (활성 직원 수), 관리자, 직원, 초대 대기 중

### 직원 테이블 컬럼
| 이름 | 부서 | 직책 | 역할 | 상태 | 마지막 로그인 | 액션 |
|------|------|------|------|------|-------------|------|
| 홍길동 (프로필 이미지 + 이름) | 영업팀 | 팀장 | 관리자 뱃지 | active | 1시간 전 | ··· 메뉴 |

**··· 메뉴 (owner만)**
- 역할 변경 (owner → manager/employee)
- 비활성화 (Soft Delete)

### 초대 대기 중 탭
| 이메일 | 역할 | 초대일 | 만료일 | 액션 |
|-------|------|--------|--------|------|
| test@co.com | 직원 | 3/11 | 3/13 | 재발송 / 취소 |

### 직원 초대 모달
- 이메일 (email)
- 역할 선택 (manager / employee)
- 「초대 발송」 버튼

## 3. 설정 화면 `/settings`

**레이아웃**: 좌측 설정 카테고리 메뉴 + 우측 콘텐츠

### 카테고리 메뉴
- 내 프로필
- 회사 정보 (owner만)
- 근무 설정 (owner만)
- GPS 출퇴근 설정 (owner만)
- 알림 설정

### 3-1. 내 프로필
- 프로필 이미지 업로드 (원형, 클릭 시 파일 선택)
- 이름, 전화번호, 부서, 직책 (편집 가능)
- 사번, 입사일 (읽기 전용)
- 비밀번호 변경 섹션 (현재 / 새 / 확인)
- 「저장」 버튼

### 3-2. 회사 정보 (owner)
- 회사명, 업종, 전화번호, 주소
- 회사 로고 업로드
- 「저장」 버튼

### 3-3. 근무 설정 (owner)
- 기본 출근 시간 (time picker)
- 기본 퇴근 시간 (time picker)
- 지각 허용 시간 (분, number input)
- 근무 요일 (월~일 토글 버튼)
- 「저장」 버튼

### 3-4. GPS 출퇴근 설정 (owner)
- GPS 활성화 toggle
- GPS 활성화 시:
  - 지도 (Kakao 또는 Google Maps embed, 핀 클릭으로 위치 설정)
  - 위도/경도 (자동 입력, 직접 수정도 가능)
  - 허용 반경 슬라이더 (50m ~ 500m, 현재: 100m)
  - 반경 표시 원이 지도에 시각화
  - 엄격 모드 toggle (반경 외 즉시 알림)
- 「저장」 버튼

### 3-5. 알림 설정
- 푸시 알림 전체 toggle
- 항목별: 업무, 메시지, 일정, 근태
- 방해 금지 시간 설정 (toggle + 시작/종료 시간)
- 「저장」 버튼
